# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-29

### Added

- Initial release
- Four counter styles: flip clock, digital, rings, and hourglass
- Celebration animation with customizable falling emoji particles
- Arrival glow effect when countdown reaches zero
- Responsive scaling using CSS Container Queries
- Grayscale mode for classic MagicMirror aesthetic
- Per-event customization for colors, icons, and celebration emojis
- Configurable display duration for passed events
